# API Endpoints

## Authentication

- **POST** `/api/auth/signup` - Register a new user
- **POST** `/api/auth/login` - Authenticate user

## Email Templates

- **GET** `/api/email-template/` - Get all email templates

## Email Sending

- **POST** `/api/send-bulk-email/` - Send bulk emails

## Logs

- **GET** `/api/logs/` - Get all email logs
